# IRON TEMPLE - Fitness Studio Website

A dark, powerful, and modern fitness studio website built with pure HTML and CSS. Features an industrial aesthetic with bold typography and smooth animations.


##  Features

- **Dark Industrial Design** - Bold black theme with electric orange accent
- **Fully Responsive** - Optimized for desktop, tablet, and mobile
- **Pure HTML/CSS** - No JavaScript required
- **Smooth Animations** - Page load animations and hover effects
- **Modern Typography** - Bebas Neue display font paired with Inter
- **Complete Sections** - Hero, about, programs, trainers, pricing, testimonials, CTA, footer

##  Design Choices

### Color Palette
- **Background:** `#0D0D0D` (Deep black)
- **Surface:** `#1A1A1A` (Dark gray)
- **Accent:** `#FF3D00` (Electric orange/red)
- **Text Primary:** `#FFFFFF` (White)
- **Text Dim:** `#999999` (Gray)

### Typography
- **Display Font:** Bebas Neue (bold, aggressive, powerful)
- **Body Font:** Inter (clean, readable)

### Why This Design?
- **Aggressive Aesthetic:** Reflects the intensity of fitness training
- **High Contrast:** Easy to read, commands attention
- **Dark Theme:** Professional, reduces eye strain
- **Bold Typography:** Matches the energy of a fitness brand
- **Motivational Copy:** Inspires action and commitment


##  Key Sections

### Hero Section
- Full-viewport height hero
- Animated title with staggered reveals
- Call-to-action button
- Stats display (500+ members, 15K hours trained, 98% success rate)

### Manifesto
- Two-column grid layout
- Featured benefits with numbered indicators
- Motivational copy about transformation

### Training Programs
- Three program cards (Power Building, Total Athlete, Endurance Elite)
- Hover effects with color accents
- Feature lists for each program

### Trainers Section
- Three trainer cards with bio
- Placeholder images (ready for real photos)
- Specialty labels

### Pricing Plans
- Three-tier pricing (Starter, Elite, Legendary)
- Featured "Most Popular" badge on middle tier
- Feature lists with checkmarks
- Call-to-action buttons

### Testimonials
- Three client testimonials
- Results displayed (weight loss, muscle gain, achievements)
- Social proof section

### Call-to-Action
- Final conversion section
- Dual CTA buttons (Book Consultation, Virtual Tour)

### Footer
- Brand info, quick links, location, hours
- Three-column layout
- Copyright notice

##  Animations

- **Page Load:** Fade-in and slide-up on hero content
- **Staggered Text:** Title lines animate in sequence
- **Hover Effects:** Cards lift and glow on hover
- **Transitions:** Smooth 0.4s transitions throughout
- **Scroll Line:** Animated scroll indicator
